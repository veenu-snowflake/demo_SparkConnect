DEMO 3 — WHAT DOES NOT WORK (04_unsupported_*.ipynb)

[Open both notebooks. Run each cell on Classic first, then Snowpark Connect]

RDDs — fails. SparkContext not in Spark Connect protocol. Use DataFrames instead.
Streaming — fails. Use Snowpipe Streaming / Dynamic Tables / OpenFlow.
MLlib — fails. Use Snowpark ML / Cortex ML.
SparkContext (broadcast, accumulator) — fails. Snowflake handles broadcasts automatically.
Repartition — silently ignored. Snowflake auto-optimizes. That is correct behavior.


WRAP-UP

Works: DataFrames, SQL, UDFs — zero code changes.
Does not work: RDDs, Streaming, MLlib — Snowflake-native alternatives exist for each.

One connection change. Everything else stays.
